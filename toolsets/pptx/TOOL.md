# PPTX Skill

## Overview
This skill enables the agent to generate PowerPoint presentations based on user requirements. It is suitable for tasks involving slide creation, editing, and exporting to .pptx format.

## Usage
- When the user requests a presentation, use this skill.
- The agent should ask for the theme, structure, and output format if not specified.
- The agent can generate an outline, and (if implemented) call a script to generate the actual PPTX file.

## Example
- "请帮我做一个关于人工智能的PPT"
- "Create a presentation about climate change"
